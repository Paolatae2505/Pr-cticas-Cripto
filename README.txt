Instrucciones de Ejecución :
    1.Ubícate en la carpeta donde se encuentra el archivo Programa.sh.
    2. En caso de no tener permisos para ejecutar el archivo, escribe en terminal :
           -> chmod +x Programa.sh
    2.Ejecuta el comando ./Programa.sh en la terminal.
    3.A continuación, se te solicitará que ingreses el nombre de dominio o la dirección IP de tu elección.
    Por ejemplo:
           -> Ingresa el nombre de dominio / IP's: example.com
    4.Se imprimirá en la terminal la información recopilada sobre el dominio o la dirección IP ingresados.
    5.Además, esta información se guardará en un archivo de texto llamado "resultados.txt" en la misma carpeta para facilitar su lectura posterior.
